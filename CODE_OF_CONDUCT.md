# Code of Conduct

TopoGPU uses a simple professional conduct policy for repository interactions.

Participants should keep discussions technical, respectful, and actionable. Harassment, personal attacks, disclosure of private information, or repeated disruptive behavior are not acceptable.

Project maintainers may moderate issues, pull requests, or discussions that violate this policy. Report conduct concerns through the repository owner contact route or a private channel listed by the project maintainers.
